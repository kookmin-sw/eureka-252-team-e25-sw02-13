    ⓐ 상상의 현실화에 관한 주제 논의
    
    ◆MIT Sixth Sense 연구팀이 만들고자 했던 것은?

      • Why? : 디지털 세계와 물리적으로 상호작용할 수 없나 

      • What? : 현실의 정보와 디지털 정보를 보다 쉽게 연동하는 장치

      • How? : 카메라와 모션인식 기술, 프로젝터 등을 통해서
    
     ◆As-Is vs To-Be는 어떻게?
    기존 컴퓨터의 입력, 출력 장치를 어떠한 방식으로 변경하였나요??

      • 키보드는? : 노트에 글씨를 쓰면 움직임과 문자를 인식

      • 마우스는? : 손의 움직임을 인식해 정해진 기능을 수행

      • 모니터는? : 몸에 달려있는 프로젝터를 통해 손이나 벽 등에 투영
  
     ◆MIT Sixth Sense Project 팀이 만든 내용이 현재에는 어떠한 장비로, 어떻게 구현이 되었을까? 

      • 현재의 기술과 약 15년 전에 MIT에서 진행한 기술의 발전 모습은? : 애플 비전 프로와 같은 증강현실(AR)을 통해 현실 공간에 디지털 정보를 겹쳐 보여주는 방식으로 구현됨
